### [멘보샤 (Menbosha)](https://github.com/modern-agile-team/Menbosha-back)

**"교내 신입생을 위한 1대1 채팅기반 멘토•멘티 서비스"** - _(2023.12.01 ~ 2024.03.17)_

`Nest.js, MySQL, Redis, TypeORM, Slack, Swagger`

- 소셜 로그인 도입으로 가입/로그인 절차를 3단계에서 1단계로 간소화하여 초기 이탈률 37% 감소
- 세션보다 인증 및 관리에 부담이 없는 JWT를 채택하여 안전하고 빠른 인증•인가 시스템 구현
- [passport strategy를 사용하여 JWT 인증로직 간소화 및 재사용성 향상](https://github.com/modern-agile-team/Menbosha-back/pull/240)
- Redis의 데이터 휘발성을 해결하기 위해 RDB (snapshot) 백업 및 복원 자동화로 데이터 영속성을 보장하여 사용자 경험 향상
- 멘토의 기여도를 측정하고 Cron을 사용해 1주일마다 인기 멘토를 선정하는 자동화 기능 구현
